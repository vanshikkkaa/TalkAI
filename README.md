# 🤖 AI Chatbot with NLP

A simple AI chatbot built using **Python** and **Natural Language Processing (NLP)**. It can understand basic user questions and reply with meaningful responses, making conversations feel more natural. This project is a great introduction to how chatbots work and how NLP can be used in real-world applications.

---

## 📌 Features

* 💬 Chat with users in real time
* 🧠 Understands basic user queries using NLP
* 👋 Responds to greetings and common questions
* ⚡ Fast and easy to use
* 🛠️ Easy to customize by adding new questions and responses

---

## 🛠️ Tech Stack

* Python
* NLTK
* JSON
* Regular Expressions (Regex)

---

## 📂 Project Structure

```text
AI-Chatbot-NLP/
│── chatbot.py          # Main chatbot program
│── intents.json        # Questions and responses
│── requirements.txt    # Required libraries
│── README.md
```

---

## 🚀 How to Run

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/AI-Chatbot-NLP.git
cd AI-Chatbot-NLP
```

### 2. Install the required libraries

```bash
pip install -r requirements.txt
```

### 3. Start the chatbot

```bash
python chatbot.py
```
